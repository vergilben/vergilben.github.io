Posted on 2019-01-28 | Post modified: 2019-02-20

写在前面：编程到底是干嘛？这是我一直在思考的问题。最近在一本书中我隐约明白了一点，书中写到，“编程的有趣之处在于将要解决的问题分成你可交给计算机完成的一组指令。编程人员就像维修工，到达现场时，会背上一些工具和配件，先看看哪里出了问题，再打开工具包，用不同的工具和配件，将各部分组合在一起从而解决问题。编程的艺术在于确定你要从工具包中取出哪些工具来解决每一部分的问题。”

Python用于数据处理

我们可以将Python本身作为数据处理者。用Python编写的程序传给Python引擎，然后Python引擎完成输出。

Python命令输入 ——> Python命令Shell ——> 结果

我们知道，Python用于处理数据，用Python编写的脚本由Python系统来解释，此后生成一些输出，我们也知道，在运行Python的计算机中，数据值由位模式表示（on和off）。
对于不同数据类型，值的存储方式和操作效果是不同的，即使为每种类型使用相同的运算符也同样如此。这是因为Python强制分离数值数据和文本数据，当Python计算数值表达式时（将数值作为结果），它仅返回数值；但如果它对字符串求值，便返回包括在引号中的文本。

1、ord函数

Python知道如何执行的一个操作是ord（ordinal value）。简单的解释它的意思就是：将一系列字符中这个字符的相应数值传给我，或者说是：给我提供表示这个字符的数值。（为什么会有一种，三分钟内我要这个人的全部信息这种感觉。。）ord函数的用法是：ord（’a’），a是给函数提供的数据。

3、bin函数

如果将计算机内存看作一串小箱子，每个小箱子都有一个唯一的数值地址，每个内存地址由8位组成。此类内存地址称为字节，每个地址的大小是一个字节。bin函数接受一个数值并返回表示该数值的位串。

可以用这bin函数来求二进制数。

使用Python内置库

1、random库

random库是可在程序中使用的随机数来源，随机数是非常有趣的一个东西，许多游戏中需要随机数，抽奖也可以用到随机数。
在random库中包含大量函数，其中randint函数生成介于特定范围的随机数。程序可给randint提供两个参数：random.randint(a,b)其中a是下限，b是上限。

time库也是非常有意思的一个库，你可以用它获得电脑时钟的日期和时间。这里主要介绍一个sleep函数，它可以使程序在一定时间段内休眠。比如让程序等待使用者10秒之后再给出答案，只是让Python程序“休息一下”。

snaps函数

不知道你在接触Python编程时有没有想过在你的程序里加入音乐，图片之类的，通过这一部分的学习，我们也许真的可以做到。

1、安装pygame库

这个过程非常简单，打开windows中的powershell或者cmd命令，输入py -m pip install pygame --user等待安装完成就好。当屏幕上出现Successfully installed xxxxxx就表明安装完成。
如果你使用的是Mac或Linux，可以使用这条命令：python3 -m pip install pygame --user
